# PNG Image Viewer with metadata
## 動作条件
Windows 11  
python 3.14 (でしか動作確認していません)
## インストール＆起動方法
start.bat を実行するだけ。  
勝手に start.bat と同じディレクトリにvenv環境が作られて実行されます。  
初回以降も start.bat を実行すればいいです。
## 使い方
### 左ペイン：ファイルブラウザ
ここにエクスプローラーに似たファイルリストが表示されます。  
マウスの進む/戻るボタンで履歴移動できます。
### 中ペイン：画像表示
ここに画像が表示されます。  
画像を右クリックすると、クリップボードへのコピーやその画像のフォルダをエクスプローラで開くことができます。
### 右ペイン：ファイル情報(メタデータ)
ここにはメタデータなどが表示されます。  
Stable Duffusion WebUIで生成された画像については
- ポジティブプロンプト
- ネガティブプロンプト
- その他情報

というように表示され、それぞれクリップボードへのコピーボタンが表示されます。  
(ComfyUIは非対応です)
